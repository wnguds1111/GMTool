# Gravity Admin Tools - 내부 대시보드 스타일 및 템플릿 분석

## 1. 개요
* **대상 사이트**: https://stage-admin.mygnjoy.com/ (로그인 후 내부 대시보드)
* **목적**: 실제 내부 대시보드 구조와 스타일을 파악하여 GMTOOL 템플릿 제작.

## 2. 레이아웃 구조 (이중 사이드바 형태)
대시보드는 **2-depth 사이드바**가 좌측에 고정된 안정적인 풀스크린 레이아웃을 사용합니다.
* **Top Header (로고 영역)**
  * 위치: 최좌측 상단 (두 사이드바의 너비만큼 차지, 약 320px)
  * 배경색: 연한 회색 (`#F3F4F6`)
  * 내용: "GRAVITY Admin Tools" 로고 및 언어 선택기 (KO)
* **1차 사이드바 (Primary Nav)**
  * 위치: 로고 영역 바로 아래 최좌측
  * 너비: 약 `80px`
  * 배경색: 생동감 있는 파란색 (`#4A7CFF` / `#3b82f6` 계열)
  * 특징: 아이콘과 영문 축약어(MEM, GAME, PAY, WEB, SET)가 세로로 나열됨. 활성화된 메뉴는 하얀색 둥근 배경에 파란 텍스트로 반전 처리.
* **2차 사이드바 (Sub Nav)**
  * 위치: 1차 사이드바 바로 우측
  * 너비: 약 `240px`
  * 배경색: 연한 회색 (`#F3F4F6`)
  * 특징: 상단에 로그인 유저명과 작은 회색 [로그아웃] 버튼. 그 아래로 펼침/닫힘이 가능한 하위 메뉴 목록 존재. 활성화된 메뉴 항목은 파란색 텍스트로 강조됨.
* **메인 콘텐츠 영역**
  * 위치: 우측 전체 (`flex: 1`)
  * 배경색: 밝은 회백색 (`#F8F9FB`)
  * 특징: 최상단에 브레드크럼(경로, 예: `계정 관리 > 계정 조회`) 표시.

## 3. 세부 컴포넌트 및 색상 디자인
* **폼(Form) 및 테이블(Table) 디자인**
  * **테이블 래퍼**: 흰색 배경, 테두리 `1px solid #E2E8F0`, 둥근 모서리(`border-radius: 4px`).
  * **라벨 셀(th/td)**: 연한 회청색 배경(`#EEF2F6`)과 굵은 검정 텍스트. 
  * **입력 셀(td)**: 흰색 배경. 
  * **입력 요소 (Input, Select, Textarea)**: 테두리가 부드럽게 둥근 형태(`border-radius: 4px`). 포커스 시 파란색 테두리로 강조. Textarea (Memo 등)는 우측에 여러 행을 병합하여 크게 배치.
* **버튼 (Buttons)**
  * 둥근 모서리 (`border-radius: 4px`). 로그인 화면과 달리 내부는 모서리가 둥글게 처리되어 현대적임.
  * **Primary Button (조회/검색)**: 짙은 파란색 배경 (`#4A7CFF`), 흰색 텍스트.
  * **Secondary Button (재입력/취소)**: 회색 배경 (`#94A3B8`), 흰색 텍스트.

## 4. GMTOOL 로컬 템플릿 제작 완료
위 분석 결과를 완벽히 반영하여 순수 HTML/CSS 만으로 동작하는 `index.html`을 재작성했습니다. 좌측 이중 사이드바 구조, 계정 조회 폼 형태, 파란색과 회색의 색상 조합, 버튼 스타일 등 실제 사이트와 거의 동일한 레이아웃을 제공합니다.
